My Personal Website

A visually rich, fully responsive personal website built with HTML, CSS, and Vanilla JavaScript to showcase my work as a Composer, Musical Director, and Software Developer.
This project blends creative storytelling, music, and web development into an immersive digital experience.

Live Demo: https://sipumzotrueman.co.za

📸 Preview

Key Features:
* Immersive Landing Experience
* Animated masked reveal loading screen
* Dynamic text animations (“WELCOME TO THE DIGITAL HOME OF…”)
* 3D rotating image carousel using transform: perspective and rotateY
* Advanced Animations & UI
* Continuous scrolling banners (skills + services)
* Dual-direction infinite carousels (10-minute seamless loops)
* IntersectionObserver-powered image flip animations
* Animated glowing borders using CSS conic gradients
* Scroll snapping for full-page section transitions
* Smart Navigation Feedback through scroll indicator with dynamic arrow visibility
* Scroll-aware UI using JavaScript event listeners
* Version control with GitHub
* Optimized for various screen sizes

Built With:
* HTML5 — Semantic structure
* CSS3 — Advanced animations, grid, flexbox, custom properties
* Vanilla JavaScript — DOM manipulation & interactivity

KEY LEARNING: 

Post-Deployment Optimization:
One of the biggest lessons from this project came after deployment, not during development.

I initially used JPEG images across the site for visuals and carousels
This led to:
Larger file sizes
Slower load times
Reduced performance (especially on mobile)

What I’m Improving:
Converting images to WebP format for:
Better compression
Faster load speeds
Implementing:
Lazy loading (loading="lazy")
Image optimization pipelines
Better asset management
